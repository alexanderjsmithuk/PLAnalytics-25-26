# Premier League 25/26 Analytics
## Summary
This project analyses player data from the 25/26 premier league season, showcasing every registered players statistics aswell as the top performers within the league.

## Tools Used
* **Data Cleaning:** Microsoft Excel
* **Database Management:** PostgreSQL / DBeaver
* **Analysis and Querying:** SQL
* **Data Visualisation:** Power BI
* **Version Control:** Github

## 📊 Interactive Dashboard Preview
![Premier League 2025/26 Attacking Efficiency Dashboard](PL_Dashboard.png)

### Key Dashboard Features:
* **KPI Summaries:** Real-time tracking of total players analyzed, total goals, and total assists.
* **Interactive Slicers:** Dynamic filtering by **Squad** and **Position (`Pos`)**.
* **Squad Contributions:** Analysis of goal contributions per 90 across top Premier League clubs.
* **Player Output Ranking:** Top performers benchmarked by combined `G+A` output per 90 minutes.
